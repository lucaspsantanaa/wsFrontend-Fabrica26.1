# Rick and Morty - Fábrica de Software UNIPÊ 2026.1

Aplicação web desenvolvida como desafio do Workshop de Front-End da Fábrica de Software do UNIPÊ 2026.1.

## Sobre o projeto

Aplicação que consome a API do Rick and Morty e exibe os personagens da série com opção de busca por nome e alternância entre visualização em grade e lista.

## Funcionalidades

- Listagem de personagens com nome, status, espécie e imagem
- Busca de personagens por nome
- Alternância entre visualização em grade e lista
- Clique na logo para voltar ao início

## Tecnologias utilizadas

- React
- JavaScript
- CSS
- Rick and Morty API

## Como rodar o projeto
```bash
npm install
npm start
```
